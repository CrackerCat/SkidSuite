# Squished Worm v2

Archived [release of SWv2 from 2014](https://hackforums.net/showthread.php?tid=4094560). 

![diagram of features](worm-diagram.png)

The default user and pass for ssh and ftp server is: `5quishusername 5quishpassword`

Libraries used:
- ftplet-api
- ftpserver-core
- mina-core
- slf4j-api
- slf4j-nop
- slf4j-simple
- sshd-core
